Local run scripts

Usage:
- On macOS/Linux: `./scripts/run-backend.sh` or `./scripts/run-frontend.sh`
- On Windows (PowerShell): `./scripts/run-backend.ps1` or `./scripts/run-frontend.ps1`

The backend scripts start the API servers (ingress + agent) on ports 8080 and 8081.
The frontend scripts run the Vite dev server for the UI (default port 5173).

Make sure you have Python 3.11+ and Node (18+) installed and a virtual environment active for Python.
